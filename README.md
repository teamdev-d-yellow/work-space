## 使い方
このテトリスゲームはカーソルとキーボードの２つの操作方法でプレイできます。
キーボード操作の場合は、以下のキーを使用してテトリミノの動きを制御してゲームを進めることができます。

### キーボード操作
- **左矢印キー**: テトリミノを左に移動
- **右矢印キー**: テトリミノを右に移動
- **下矢印キー**: テトリミノを下に移動
- **上矢印キー**: テトリミノを回転（時計回り）
- **スペースキー**: テトリミノを交換
